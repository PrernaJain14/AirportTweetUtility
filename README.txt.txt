1.Install mysql database
2.Import AirportTweetUtility in IDE as Maven project(I used eclipse oxygen)
3.Change the mssql credentials in application.properties
4.Change the twitter credentials in twitter.properties
5.create the table in your schema from attached ddl AirportTweet.sql
6.Import the refrence data from airport.csv 
7.Change the property character_set_server=utfmb4 in my.ini/my.cnf file. (to support utf-8 4 byte data)
8.run the program.


For Info:

I am adding table's ddl for the reveiw of table structure in Airport_tweet_ddl.
Also,there is .csv file for each table so that you can examine the data I am improting in tables.
